# At-Home Lab (ISSO Preparation)

This repository documents a structured home lab used to develop and demonstrate
Information System Security Officer (ISSO) competencies.

## Objectives
- Linux and Windows audit logging
- DNS security monitoring
- Time synchronization and log integrity
- NIST 800-53 control familiarity
- Evidence-based documentation practices

## Structure
- architecture/ – design decisions and diagrams
- phase-1-linux-logging/ – Linux audit and time logging
- phase-1-5-dns-filtering/ – Pi-hole DNS monitoring
- phase-2-windows-event-logs/ – Windows Security logs
